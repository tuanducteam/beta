In this article I would like to share with you the list of CSS Frameworks that support Website design.

{% include image.html src="css-frameworks.jpg" webp="css-frameworks.webp" type="jpg" alt="List of awesome CSS frameworks" %}

## Base / Reset / Normalize

- **{% include external-link.html href="https://necolas.github.io/normalize.css/" title="normalize.css" %}** - Modern, HTML5-ready alternative to CSS resets.
- **{% include external-link.html href="https://csstools.github.io/sanitize.css/" title="sanitize.css" %}** - The best-practices CSS foundation.
- **{% include external-link.html href="https://jgthms.com/minireset.css/" title="minireset.css" %}** - Tiny modern CSS reset.
- **{% include external-link.html href="https://github.com/inuitcss/inuitcss" title="inuitcss" %}** - Extensible, scalable, Sass-based, OOCSS framework for large and long-lasting UI projects.
- **{% include external-link.html href="https://github.com/hankchizljaw/modern-css-reset" title="modern-css-reset" %}** - Tiny little reset that you can use as the basis of your CSS projects.
- **{% include external-link.html href="https://github.com/filipelinhares/ress" title="ress" %}** - Modern CSS reset.
- **{% include external-link.html href="https://github.com/sindresorhus/modern-normalize" title="modern-normalize" %}** - Normalize for Chrome, Firefox, and Safari only (no support for Edge or Internet Explorer).
- **{% include external-link.html href="https://github.com/frontaid/natural-selection" title="Natural Selection" %}** - Collection of best-practice CSS selectors.

## Class-less

Frameworks that use semantic HTML and do not rely on classes.

- **{% include external-link.html href="https://watercss.kognise.dev/" title="Water.css" %}** - Just-add-CSS collection of styles to make simple websites just a little nicer.
- **{% include external-link.html href="https://andybrewer.github.io/mvp/" title="MVP.css" %}** - Minimalist stylesheet for HTML elements.
- **{% include external-link.html href="https://oxal.org/projects/sakura/" title="sakura" %}** - Minimal classless CSS framework/theme.
- **{% include external-link.html href="https://mblode.github.io/marx/" title="Marx" %}** - Classless CSS reset.
- **{% include external-link.html href="https://yegor256.github.io/tacit/" title="Tacit" %}** - CSS framework for dummies, without classes.
- **{% include external-link.html href="https://igoradamenko.github.io/awsm.css/" title="awsm.css" %}** - CSS library for semantic HTML markup without classes, attributes, etc.

## Very Lightweight

Frameworks that are smaller than ~5KB.

- **{% include external-link.html href="https://purecss.io" title="Pure" %}** - Set of small, responsive CSS modules that you can use in every web project.
- **{% include external-link.html href="https://milligram.io" title="Milligram" %}** - Minimalist CSS framework.
- **{% include external-link.html href="https://picnicss.com" title="Picnic CSS" %}** - Lightweight and beautiful library.
- **{% include external-link.html href="https://jenil.github.io/chota/" title="Chota" %}** - Really small CSS framework.

## General Purpose

- **{% include external-link.html href="https://getbootstrap.com" title="Bootstrap" %}** - The most popular HTML, CSS, and JavaScript framework for developing responsive, mobile first projects on the web.
- **{% include external-link.html href="https://bulma.io" title="Bulma" %}** - Modern CSS framework based on Flexbox.
- **{% include external-link.html href="https://get.foundation/" title="Foundation" %}** - The most advanced responsive front-end framework in the world.
- **{% include external-link.html href="https://getuikit.com" title="UIkit" %}** - Lightweight and modular front-end framework for developing fast and powerful web interfaces.
- **{% include external-link.html href="https://picturepan2.github.io/spectre/" title="Spectre.css" %}** - Lightweight, responsive and modern CSS framework.
- **{% include external-link.html href="https://primer.style/" title="Primer" %}** - CSS framework that powers GitHub's front-end design.
- **{% include external-link.html href="https://www.carbondesignsystem.com/" title="Carbon Components" %}** - Component library behind IBM's Carbon Design System.
- **{% include external-link.html href="https://fomantic-ui.com/" title="Fomantic-UI" %}** - Build beautiful websites fast, with concise HTML, intuitive javascript, and simplified debugging.
- **{% include external-link.html href="https://www.blazeui.com" title="Blaze UI" %}** - Open source modular toolkit providing great structure for building websites quickly with a scalable and maintainable foundation.
- **{% include external-link.html href="https://www.patternfly.org/" title="PatternFly" %}** - UI framework for enterprise web applications.
- **{% include external-link.html href="https://getbase.org" title="Base" %}** - Rock solid, responsive HTML/CSS framework.
- **{% include external-link.html href="https://turretcss.com" title="turretcss" %}** - Responsive front-end framework for accessible and semantic websites.
- **{% include external-link.html href="https://spiderpig86.github.io/Cirrus/" title="Cirrus" %}** - Responsive and comprehensive CSS framework with beautiful controls and simplistic structure.
- **{% include external-link.html href="https://jonathanharrell.github.io/hiq/" title="HiQ" %}** - Simple CSS foundation with responsive typography and input styling.

## Material Design

- **{% include external-link.html href="https://material.io/components/web/" title="Material Components Web" %}** - Modular and customizable Material Design UI components for the web.
- **{% include external-link.html href="https://materializecss.com" title="Materialize" %}** - CSS framework based on material design.
- **{% include external-link.html href="https://www.muicss.com" title="MUI" %}** - Lightweight CSS framework that follows Google's Material Design guidelines.

## Utility-based

- **{% include external-link.html href="https://tailwindcss.com" title="Tailwind CSS" %}** - Utility-first CSS framework for rapid UI development.
- **{% include external-link.html href="https://tachyons.io" title="Tachyons" %}** - Quickly build and design new UI without writing CSS.

- **{% include external-link.html href="https://nostalgic-css.github.io/NES.css/" title="NES.css" %}** - NES-style CSS Framework.
- **{% include external-link.html href="https://jdan.github.io/98.css/" title="98.css" %}** - CSS library for building interfaces that look like Windows 98.

- **{% include external-link.html href="https://edwardtufte.github.io/tufte-css/" title="Tufte CSS" %}** - Style your webpage like Edward Tufteâ€™s handouts.

- **{% include external-link.html href="https://github.com/vinibiavatti1/TuiCss" title="TuiCss" %}** - Library to create MS-DOS interfaces.

## Toolkits

Toolkits are closer to a library than to a framework.
They do not provide actual styling but utilities (e.g. mixins) that you can use in your CSS preprocessor.

- **{% include external-link.html href="https://www.bourbon.io/" title="Bourbon" %}** - Lightweight Sass tool set.

Frameworks that have not seen any real development for more than about one year or officially have been deprecated.
